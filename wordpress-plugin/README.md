# WordPress Plugin

`happiness-manager/` がWordPress用プラグイン本体です。

開発中の初版なので、まずはテスト用WordPressで動作確認してから本番サイトへ入れてください。

ブラウザ内の一時WordPressで試す場合は、WordPress Playgroundを使えます。
